Es un template engine.

¿Qué son los controladores?

*****************************
10.

Es un mecanismo que usa Laravel para inyectar las dependencias de objetos o métodos.

11.

Es un archivo en el que podemos establecer las variables de entorno para nuestro proyecto.

12.

Para migrar datos de una DB (base de datos) a otra.

*****************************

Es la interfaz de línea de comandos incluida por Laravel para ayudarnos mientras creamos una app.

15.

Es una técnica de programación para convertir registros de una base de datos en objetos.

16.

Es un ORM incluido por Laravel.
